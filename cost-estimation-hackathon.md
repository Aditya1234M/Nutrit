## 12. Hackathon-Specific Optimizations

### Cost-Saving Strategies

**1. Use Free Tiers Everywhere**
- Railway.app for backend (free)
- Supabase for database + auth + storage (free)
- Cloudinary for images (free)
- Google Colab for AI training (free)
- **Total: ₹0/month**

**2. Leverage Student Benefits**
- GitHub Student Pack
- Google Cloud credits
- AWS Educate credits
- **Saves: ₹50,000+ in credits**

**3. Optimize AI Model**
- Use pre-trained models (EfficientNet)
- Train once, use forever
- No need for continuous retraining
- **Saves: ₹830/month (Colab Pro)**

**4. Delay App Store Launch**
- Use TestFlight for iOS
- Use Internal Testing for Android
- Launch to stores only after validation
- **Saves: ₹688/month**

**5. Use Serverless Architecture**
- No always-on servers
- Pay only for actual usage
- Perfect for low traffic
- **Saves: ₹415-2,000/month**
